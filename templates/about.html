About Wasabi Cloud Storage Bot

How It Works

This service uses Telegram as a frontend interface and Wasabi's S3-compatible storage for secure file storage.

Technology Stack:

  • Python with Pyrogram for Telegram bot
  • Flask for web interface
  • Wasabi Hot Cloud Storage for file storage
  • HTML5 media players for video/audio playback

Privacy & Security

Your files are stored securely in Wasabi cloud storage with the following protections:

  • Files are stored in private buckets
  • Download links expire after 24 hours
  • No personal data is stored on the web server
